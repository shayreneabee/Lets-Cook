# Let's Cook Y'all Photography Library

This folder is the production home for brand photography.

Style direction:
- Bright natural daylight
- Warm color temperature
- Clean modern kitchens
- Real people cooking
- Diverse families, cooks, and skill levels
- Hands-on cooking techniques
- Shallow depth of field
- Commercial culinary education quality

Avoid:
- Watermarked images
- Dark or muddy photos
- Empty kitchen shots
- Repeated generic stock
- AI-looking faces
- Overly staged models

Use `image-library.json` as the slot map. Keep filenames stable so the app can switch from current safe fallbacks to final photography without changing page code.
